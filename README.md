# Restriction Adult Content On Screen

## Overview
SafeScreen AI is a frontend prototype of a parental control system designed to restrict adult content displayed on screens. It demonstrates how inappropriate content can be detected and blocked through a monitoring dashboard.

## Features
- Monitoring Status Dashboard
- Adult Content Detection Simulation
- Warning Popup System
- Blocked Content Counter
- Responsive User Interface

## Technologies Used
- HTML
- CSS
- JavaScript
